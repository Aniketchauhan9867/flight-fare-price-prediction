# flight-fare-price-prediction

## Author

Aniket Chauhan

## Project Overview

This project aims to predict flight ticket prices using historical flight data and machine learning techniques. The analysis considers factors such as airline, journey date, source, destination, duration, and number of stops.

## Objectives

* Analyze factors affecting airfare prices.
* Perform data cleaning and preprocessing.
* Build machine learning models to predict ticket prices.
* Identify key variables influencing fare prices.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-Learn

## Project Workflow

1. Data Cleaning
2. Feature Extraction
3. Exploratory Data Analysis (EDA)
4. Data Preprocessing
5. Model Building
6. Model Evaluation

## Features Considered

* Airline
* Source
* Destination
* Total Stops
* Journey Date
* Duration
* Departure Time
* Arrival Time

## Machine Learning Models Used

* Linear Regression
* Decision Tree Regressor
* Random Forest Regressor

## Evaluation Metrics

* R2 Score
* Mean Absolute Error (MAE)
* Root Mean Squared Error (RMSE)

## Business Impact

The model can assist travelers in estimating ticket prices and support travel companies in understanding pricing trends and customer booking behavior.

## Conclusion

The project successfully developed models capable of predicting flight fares and identified important factors that influence airfare pricing.
